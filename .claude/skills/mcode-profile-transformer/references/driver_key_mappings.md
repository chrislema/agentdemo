## Usage Example

For "Identify Potential" motivation:
```json
"drivers": {
  "dig": "Can see beyond surface qualities to underlying potential",
  "search": "Motivated when finding value in places others have overlooked",
  "retrieve": "Energy focused on drawing out and realizing possibilities",
  "reveal": "Enjoys bringing discoveries to light and seeing them used by others"
}
```

**Important**: Personalize driver descriptions using language from the individual's report, not generic definitions.
